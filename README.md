# Sales Executive Performance Dashboard - Excel

Interactive Excel dashboard tracking sales executive performance across multiple cities. Built following Satish Dhawale's YouTube tutorials, this dashboard analyzes sales targets vs actuals with city-wise filtering.

# Dashboard Features

- **Performance Variance**: Visual indicators showing over/under performance
- **City-wise Slicer**: Filter data by city availability for regional analysis
- **Dynamic Charts**: Pivot charts updating with slicer selections
- **KPIs**: Total sales, achievement percentage

# How to Use

1. Open `Sales-Dashboard.xlsx`
2. Use **City Slicer** (top-left) to filter by location
3. Analyze executive performance metrics and charts
4. Identify top performers and underperformers instantly

# Business Insights Provided

- Which executives are exceeding targets
- City-wise sales performance gaps
- Regional trends and opportunities
- Quick performance ranking across teams
